# FIFISHING

## サイト概要
### サイトテーマ
実際に自分が釣りに行った場所を投稿できるSNSサイト

### テーマを選んだ理由
釣りを始めたばかりの人が誤って釣り禁止エリアに行ってしまうことがあるので、おすすめの釣りスポットを探すことができ、既に釣りをしている人たちも、もっといい場所を知ることができるサービスがあれば便利だと考え、このテーマにしました。

### ターゲットユーザ
* 釣りが好きな人
* 違う場所に行きたい人
* これから釣りを始めようと思っている人

### 主な利用シーン
いつもと違う釣りのスポットに行きたいとき、他の人がどのような場所に行っているのかを知りたい場面で利用する
